# API package

`api/` serves normalized data products and ML-oriented endpoints.

## Responsibilities
- Load normalized Parquet/JSON artifacts generated by `pipeline/`.
- Expose query endpoints for cleaned match/player records.
- Expose clustering and prediction endpoints for frontend analysis workflows.
- Enforce payload validation against shared schemas in `shared/`.

## Suggested endpoints
- `GET /matches`
- `GET /players/{player_id}`
- `POST /cluster`
- `POST /predict`
