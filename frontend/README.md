# Frontend package

`frontend/` contains the D3.js visual analytics application.

## Responsibilities
- Render interactive charts backed by API responses.
- Support match- and player-level exploration.
- Visualize derived metrics and cluster segmentation.

## Suggested components
- Time-series plots for Elo trend and win rate.
- Scatter/cluster views for player segmentation.
- Drill-down panels for match and breakpoint detail.
